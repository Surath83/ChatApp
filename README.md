<h1 align="center">💬 ChatApp - Real-Time Messaging Application</h1>

<p align="center">
  A full-stack real-time chat application built using modern web technologies.

<hr>

<h2>📌 Overview</h2>
<p>
ChatApp is a real-time messaging platform designed to enable seamless communication between users.
It provides instant messaging capabilities with a responsive UI and efficient backend handling.
The project demonstrates full-stack development skills including frontend, backend, and database integration.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>🔐 User Authentication (Login/Register)</li>
  <li>💬 Real-time messaging</li>
  <li>📡 WebSocket / Socket-based communication</li>
  <li>👥 One-to-one chat system</li>
  <li>📱 Responsive UI design</li>
  <li>⚡ Fast message delivery</li>
  <li>🗂️ Persistent chat storage (if implemented)</li>
</ul>

<h2>⚙️ Installation & Setup</h2>

<h3>1️⃣ Clone the Repository</h3>
<pre>
git clone https://github.com/Surath83/ChatApp.git
cd ChatApp
</pre>

<h3>2️⃣ Install Dependencies</h3>

<p><b>Backend:</b></p>
<pre>
cd backend
npm install
</pre>

<p><b>Frontend:</b></p>
<pre>
cd frontend
npm install
</pre>

<h3>3️⃣ Setup Environment Variables</h3>
<p>Create a <code>.env</code> file in the backend folder:</p>

<pre>
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
</pre>

<h3>4️⃣ Run the Application</h3>

<p><b>Start Backend:</b></p>
<pre>
cd backend
npm start
</pre>

<p><b>Start Frontend:</b></p>
<pre>
cd frontend
npm start
</pre>
